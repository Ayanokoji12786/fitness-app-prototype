# Fitness App Prototype

A fitness tracking app prototype covering meal logging, workout plans, body metrics charts, nutrition insights, and a community feed. Built primarily as a deployment and debugging exercise: getting a React-based app running end-to-end and installed on a real Android device, not just in a browser tab.

## Why this exists

This was built to learn the unglamorous half of shipping software: packaging a web app into an APK, debugging errors that only show up on a real device, and getting something installable, before taking on a more complex project.

## Features

- Onboarding flow (basic info, goals, metrics)
- Daily meal logging and nutrition summary
- Workout plan and consistency tracking
- Body metrics and wellness charts
- Community feed
